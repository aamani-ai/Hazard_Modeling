# Notebooks

Exploratory and analysis notebooks for hazard / loss modeling.

- Use the project environment: `source ../.venv/bin/activate`, then register a kernel:
  `python -m ipykernel install --user --name hazard_modeling`.
- Keep heavy data out of git (see [`../.gitignore`](../.gitignore)) — read from shared/cloud
  sources or `docs/` instead of committing large files.
